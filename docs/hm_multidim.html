Multidimensional Histogram


For a multidimensional case the main definitions of the histogram model stay the same as shown here. So, we have the universal set, UUU, and elements of a histogram correspond to its elements. However, now each element x∈Ux \in Ux∈U is a vectors (x1,⋯ ,xm)(x_{1}, \cdots , x_{m})(x1​,⋯,xm​), and UUU is the Cartesian product over mmm sets U1,⋯ ,UmU_1,\cdots, U_mU1​,⋯,Um​, denoted U1×⋯×UmU_1 \times \cdots \times U_mU1​×⋯×Um​, where mmm is the number of dimensions, UiU_iUi​ is the universal set of the iiith dimension and xix_ixi​ belongs to UiU_iUi​.

Histogram of a data instance can be represented as

H(d)=(h(e1∣d),⋯ ,h(en∣d)∣0≤h(x∣d)≤1,∑x∈Uh(x∣d)=1),H(d)=\left( h(e_1 \mid d), \cdots, h(e_n \mid d) \mid 0 \leq h(x \mid d) \leq 1, \sum_{x\in U}h(x \mid d) = 1 \right),H(d)=(h(e1​∣d),⋯,h(en​∣d)∣0≤h(x∣d)≤1,x∈U∑​h(x∣d)=1),

where ei=(e1,i1,⋯ ,em,im)e_i = (e_{1,i_1}, \cdots , e_{m,i_m})ei​=(e1,i1​​,⋯,em,im​​) and ej,ik∈Uje_{j,i_k} \in U_{j}ej,ik​​∈Uj​

The set of all high-level elements is defined as

EG={X∣X⊆U}, E^G=\left\{X\vert X \subseteq U\right\}, EG={X∣X⊆U},

where XXX can be represented as follows

X={(e1,⋯ ,em)∣ej∈Uj}⊆UX = \{ (e_1, \cdots, e_m) \mid e_j \in U_{j} \} \subseteq UX={(e1​,⋯,em​)∣ej​∈Uj​}⊆U

In a general form we can write it as

X=(X1,i1,⋯ ,Xm,im),X = (X_{1,i_1}, \cdots , X_{m,i_m}), X=(X1,i1​​,⋯,Xm,im​​),

where Xj,ij⊆UjX_{j,i_j} \subseteq U_jXj,ij​​⊆Uj​.

And for the case when we use the Cartesian product of universal sets that each of them represents a single dimension (or a data feature), XXX can be defined as:

X={(e1,⋯ ,em)∣ej∈Xj,ij}⊆U.X = \{ (e_1, \cdots, e_m) \mid e_j \in X_{j,i_j} \} \subseteq U.X={(e1​,⋯,em​)∣ej​∈Xj,ij​​}⊆U.

The general form will be used later for querying as it is a more convenient and natural way to represent high-level elements for each dimension (see an image retrieval example below).


Histogram Algebraic System


Histogram algebraic system for the multidimensional case is the same as for the one-dimensional case (see here). There is no difference in definitions of the carrier and operations, but here we refer to multidimensional UUU:
